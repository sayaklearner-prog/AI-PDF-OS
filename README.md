## 🌟 Overview

**PDF OS 2.0** is an AI-powered document operating system designed to completely reinvent how people interact with documents.

Instead of simply viewing PDFs, users can:

✅ Analyze contracts with AI

✅ Detect legal and business risks

✅ Generate executive summaries

✅ Chat with documents in natural language

✅ Edit PDFs professionally

✅ Add signatures and annotations

✅ Create trackable QR campaigns

✅ Collaborate with an intelligent document copilot

Built with modern AI architecture, PDF OS combines the intelligence of ChatGPT, the editing power of professional PDF tools, and the flexibility of a modern SaaS platform.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn
- Apryse WebViewer License Key (Trial or Production)

Get your free key:

https://dev.apryse.com/

---

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/pdf-os.git

cd pdf-os
```

---

### 2️⃣ Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

---

### 3️⃣ Configure Environment Variables

Create:

```bash
.env
```

Add required API keys:

```env
AI_ML_API_KEY=your_key
FEATHERLESS_API_KEY=your_key
APRYSE_LICENSE_KEY=your_key
```

---

### 4️⃣ Start Development Server

```bash
npm run dev
```

---

### 5️⃣ Open Application

```text
http://localhost:3000
```
